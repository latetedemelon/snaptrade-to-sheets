SnapTrade Add-on Help

  1. Set API Keys: Open Settings → Configure API Keys and save your Client ID and Consumer Key.
  2. Register User: Use Settings → Register User to create a userId and userSecret stored in your account.
  3. Connect Brokerage: Use Connect Brokerage to open the SnapTrade portal and link your accounts.
  4. Import Data: Refresh holdings, balances, or transactions from the SnapTrade menu to populate sheets.
  5. Need Help? Review README instructions or contact your SnapTrade administrator.
